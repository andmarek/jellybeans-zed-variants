# Jellybeans Variants

A port of [WTFox's Jellybeans](https://github.com/WTFox/jellybeans.nvim) color schemes for [Zed](https://zed.dev).

## Themes

- **Jellybeans** — dark
- **Jellybeans Light** — light
- **Jellybeans Mono Light** — light, monochromatic

## Installation

Search for "Jellybeans Variants" in Zed's extension manager, or run `zed: extensions` from the command palette.
